# DataDumper.Temp

Utilitaires de dump et de comparaison de donnees .NET : diff JSON d'objets, export/import et tests.

## Structure

| Fichier / Dossier | Role |
|-|-|
| `Tools.DataDumper.Compare/` | Comparaison d'objets via ObjectDiffPatch |
| `Tools.DataDumper.Console/` | Application console pour le dump |
| `Tools.DataDumper.Core/` | Logique principale d'export/import |
| `Tools.DataDumper.Tests/` | Tests unitaires |
